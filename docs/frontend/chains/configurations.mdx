# Chain Configurations

This is a list of all available chain configurations. If you want to add a custom chain,
you can extend [Development](/frontend/chains/configurations#development) or
[Custom](/frontend/chains/configurations#custom). Note that the chain ids for each or them
must be `development` and `custom` respectively.

If you would like to add another chain please create a PR and add it
[here](https://github.com/paritytech/useink/blob/main/chains/data/chaindata.ts) for value
bearing chains, and
[here](https://github.com/paritytech/useink/blob/main/chains/data/testdata.ts) for
testnets.
