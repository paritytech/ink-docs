En este portal buscamos proveer de documentación exhaustiva para el desarrollo de cualquier proyecto
con ink!.

En caso de necesitar un tutorial paso a paso dedicado para principiantes, por favor dirijirse a nuestro 
[Tutorial Guiado](https://docs.substrate.io/tutorials/smart-contracts/).

De encontrar errores en la documentación provista, o si ciertas áreas son poco claras, por favor déjanoslo
saber, ya sea <a href="https://github.com/paritytech/ink-docs/issues">creando un issue</a>
o <a href="https://github.com/paritytech/ink-docs/pulls">haciendo un pull-request</a>.

Nuestro objetivo es proveer buena documentación para que los desarrolladores sean lo más productivos posible.
